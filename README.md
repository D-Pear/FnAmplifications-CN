# FN Amplifications - FN科技

由FN-FAL113编写的4合1附属插件，添加了发电机、材料发电机、机器以及太阳能发电机

感谢 Jeff(LiteXpansion) 和 Walshy(Slimefun) 提供的工具包  
感谢 waleks 的 SMG 附属 (现在由 Sefiraat 与 Variananora 维护)

本插件支持1.15-1.18版本
